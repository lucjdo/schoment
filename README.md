# Schoment: The School Management

[![Deployment Pipeline](https://github.com/lucjdo/schoment/actions/workflows/pipeline.yml/badge.svg)](https://github.com/lucjdo/schoment/actions/workflows/pipeline.yml)

For running the app just clone the repo and run:
- `npm run install`
- `npm run dev`

The app uses session storage as its 'backend' to fetch and save rooms and students. You don't need to worry because all data will be erased after you close the window.

## Stack
- React + Vite
- Typescript
- React query
- Context
- Material UI
- EsLint
